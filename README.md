# Catagotchi
Файлы для просмотра работы проекта на Java для новичков кто работает с Java/SQL по типу Tamagotchi
Как использовать:
1)Для пользования разархивируйте, чтобы файл нашёл картинки и вы не тратили свои нервы.
2)Если хотите иметь сохранения:
  1)Установите Denwer
  2)Создать базу данных с именем 'catagotchi'
  3)Импортируйте SQL файл в 'catagotchi' в PHPMyAdmin
  4)Пользуйтесь
